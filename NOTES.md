Notes:

* If want to disable `int` parameter encoding, define `DD_DISABLE_ENCODE_INT` before including `dumbdisplay.h`, like

  ```
  #define DD_DISABLE_ENCODE_INT
  #include "dumbdisplay.h"
  DumbDisplay dumbdisplay(new DDInputOutput());
  ```
